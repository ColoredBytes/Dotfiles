# :fox_face: Firefox Extensions

**These are all the Firefox extensions I use.**

- [Amazon Container by krober](https://addons.mozilla.org/en-US/firefox/addon/contain-amazon/)
- [Bitwarden Password Manager by Bitwarden Inc.](https://addons.mozilla.org/en-US/firefox/addon/bitwarden-password-manager/)
- [Decentraleyes by Thomas Rientjes](https://addons.mozilla.org/en-US/firefox/addon/decentraleyes/)
- [uBlock Origin by Raymond Hill](https://addons.mozilla.org/en-US/firefox/addon/ublock-origin/)
- [Google Container by containers-everywhere](https://addons.mozilla.org/en-US/firefox/addon/google-container/)
- [New Tab Override by Sören Hentzschel](https://addons.mozilla.org/en-US/firefox/addon/new-tab-override/)
- [SimpleLogin: Open-source Email Protection by SimpleLogin & Proton Pass team](https://addons.mozilla.org/en-US/firefox/addon/simplelogin/?utm_source=addons.mozilla.org&utm_medium=referral&utm_content=search)
- [Startpage by Startpage](https://addons.mozilla.org/en-US/firefox/addon/startpage-private-search/)
- [Twitter Container by Vishwa](https://addons.mozilla.org/en-US/firefox/addon/twitter-container/)
