## Instructions

- Clone the repo.
```
git clone https://github.com/ColoredBytes/Dotfiles.git
```
- Make scripts Executable
```
find /Dotfiles -type f -name "*.sh" -exec chmod +x {} \;
find /Dotfiles -type f -name "*.zsh" -exec chmod +x {} \;
```
